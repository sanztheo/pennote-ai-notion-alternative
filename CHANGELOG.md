# Changelog

All notable changes to Pennote will be documented in this file.

## [1.0.0] - 2025-01-07

### Initial Release

Pennote is officially live! The AI-powered note-taking app built for students.

### Features

#### Note Editor
- Notion-like block editor with slash commands
- Markdown support
- LaTeX equations (inline and block)
- Mermaid diagrams
- Code blocks with syntax highlighting
- Real-time collaboration

#### AI Assistant
- Ask questions about your notes
- Generate summaries
- Get explanations
- Research from Wikipedia, PDFs, and the web

#### Smart Quizzes
- Auto-generate quizzes from your notes
- Multiple choice, True/False, Open questions
- Adaptive difficulty
- Detailed corrections with source references
- Progress tracking

#### Organization
- Workspaces
- Projects and nested pages
- Page mentions and links
- PDF import

#### Dashboard
- Learning statistics
- Strengths and weaknesses analysis
- Revision streaks
